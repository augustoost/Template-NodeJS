Este repositório tem como objetivo ajudar na criação de projetos NodeJS.

O template contém:

1- Estrutura de pastas padrão MVC.
2- Express
3- Nodemon
4- Docker
5- sequelize
6- Config inicial do Postgres
7- Sentry
8- Arquivo .env.example
9- Youch
10- eslint (airbnb-base)
11- prettier
12- editorconfig
13- Debug (VSCode)
